Code agreements

Naming styles:
1 - HelloWorld
2 - helloWorld
3 - hello_world
Global variables, class/struct public vars	1
Class/struct private/protected vars		2
Local vars, func/methods arguments		3
Functions					1
All class/struct methods			1
Class/Struct/Namespace names			1 (without S/C prefix)

Pointers/references - not Class &var, but Class& var.

Code editor - use real tabs, not spaces.

Brackets
not
	if(...){
		...
	}
but
	if(...)
	{
		...
	}

Expression per line:
not
	x=y; y++; x*=2;
but
	x=y;
	y++;
	x*2;

Type names
signed
	1 char
	2 short
	4 int
	8 __int64
unsigned
	1 BYTE
	2 WORD
	4 DWORD
other
	bool
	void
